rmt-denoise v2.3.0

Image denoising via Random Matrix Theory. Two denoisers — MP-Law thresholding (i.i.d. Gaussian) and GeneralizedCov (oracle search for (a, β) via differential evolution). The Python library counterpart to EigenDenoise.

cairometal NEW

pycairo-compatible 2D vector graphics rendered on the Apple GPU via Metal — 246 C functions, drop-in import cairo_metal as cairo, pixel-diffed against real cairo. Ships a macOS arm64 wheel (Python ≥3.10). The first GPU/Metal cairo on PyPI.
